ReSC Multiglobe
---------------

This software is designed for visualising multiple environmental datasets simultaneously.

To run, please double-click the resc-multiglobe.jar file, or run it with the command:

java -jar resc-multiglobe.jar

The software comes with example datasets taken from:

http://www.esrl.noaa.gov/psd/data/gridded/

For usage and configuration please see the included quickstart guide.

More information about this software can be found at:

http://www.resc.rdg.ac.uk/videowall.php
